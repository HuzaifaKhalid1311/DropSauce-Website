# DropSauce Website

Static website for [DropSauce](https://github.com/HuzaifaKhalid1311/DropSauce).

## Local preview

Run a static server from this directory:

```powershell
python -m http.server 4173
```

Then open `http://localhost:4173`.

## Pages

- `/` - project introduction and app tour
- `/privacy` - privacy policy
- `/terms` - terms of service
