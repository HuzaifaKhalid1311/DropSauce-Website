# drop-sauce.app

Official website for **DropSauce**, a free and open-source manga reader for Android.

Live at <https://drop-sauce.app>.

- `index.html` — homepage / app gateway
- `privacy.html` — Privacy Policy (covers Google user data for the Google Drive sync feature)
- `terms.html` — Terms of Service
- `style.css` — shared responsive design system
- `script.js` — mobile navigation and reveal animations
- `media/` — app screenshots and icon artwork
- `icon.svg` — favicon / app mark
